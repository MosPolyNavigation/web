# Политех Навигация

Веб-приложение навигации по кампусу Московского Политеха.

Стек: `React 19` + `TypeScript` + `Vite`.

## Локальный запуск

### 1. Установить зависимости

```bash
npm ci
```

### 2. Запустить dev-сервер

```bash
npm run dev
```

По умолчанию Vite пытается стартовать на `http://localhost:3000/`.
Если порт занят, он автоматически выберет следующий свободный порт и покажет его в консоли.

### 3. Открыть приложение в браузере

Откройте адрес, который вывел Vite, например:

```text
http://localhost:3000/
```

## Переменные окружения

Для локальной разработки `.env` не обязателен.
Если нужно переопределить настройки, можно создать его на основе примера:

```bash
cp example.env .env
```

Доступные переменные:

- `USE_HTTPS=true` включает HTTPS только если в корне проекта есть файлы `localhost-key.pem` и `localhost.pem`.
- `VITE_BASE_PATH=` базовый путь приложения. Для локальной разработки обычно оставляется пустым.
- `VITE_BASE_PATH=/web/` используется для публикации на `gh-pages`.

## Dev override для времени расписания

В локальной разработке можно подменить текущее время для карточки расписания через query-параметр `debugTime`.

Пример:

```text
http://localhost:3000/?debugTime=11:35
```

Поддерживается формат `HH:mm`.
Этот override работает только в dev-режиме и не действует в production-сборке.

## Dev override для баннеров

В локальной разработке можно принудительно показать все баннеры приложения через query-параметр `showBanners=1`.

Пример:

```text
http://localhost:3000/?showBanners=1
```

Этот override работает только в dev-режиме и игнорирует обычные условия показа баннеров.

Параметры можно комбинировать:

```text
http://localhost:3000/?showBanners=1&debugTime=11:35
```

## Dev override для платформы PWA

В локальной разработке можно подменить платформу для экрана установки PWA через query-параметр `debugPlatform`.

Поддерживаются значения:

- `ios`
- `android`
- `desktop`

Примеры:

```text
http://localhost:3000/?debugPlatform=ios
http://localhost:3000/?showBanners=1&debugPlatform=android
```

Этот override работает только в dev-режиме.

## Баннеры

Баннеры отображаются стеком внутри окна приложения. Если активно несколько баннеров сразу, они выстраиваются друг над другом.

| Баннер | Назначение | Обычные условия показа | Когда скрывается |
| --- | --- | --- | --- |
| `PwaInstallBanner` | Предлагает установить приложение как PWA | Показывается на экране карты (`PLAN`), если приложение ещё не открыто как установленное PWA, браузер не Firefox и локальное хранилище разрешает показ | После закрытия баннер прячется на 2 дня. После успешной установки прячется полностью для этого браузера |
| `EventInfoBanner` | Сообщает о событии дня и ведёт в поиск | Показывается на экране карты (`PLAN`) только `2026-04-07` по Москве, только в кампусе `БС / BS / Большая Семёновская`, только если в этом кампусе есть event-аудитории | После закрытия прячется до конца дня по Москве |

В dev-режиме `showBanners=1` принудительно показывает все баннеры и игнорирует обычные условия показа.

```bash
npm run dev        # локальная разработка
npm run build      # production build в dist/
npm run host       # локальный preview собранной версии
npm run hostshare  # preview на 3000 порту с --host

```
